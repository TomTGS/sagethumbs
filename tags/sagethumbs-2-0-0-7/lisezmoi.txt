Fonctionnalités:

* Affichage miniature étendu en mode dossier dans l'explorateur
* Miniature dans le menu contextuel de l'explorateur (clic droit)
* Bulle d'information étendue
* Support de 131 formats d'image via la bibliothèque GFL
* Support de 26 formats supplémentaires via les plugins de XnView (si installés)
* Envoi par courriel
* Conversion en GIF,JPG,BMP en un clic de souris
* Sélection comme papier peint
* Copie dans le presse-papiers

Veuillez lire attentivement l'information ci-dessous. 
	SageThumbs étant une extension du menu contextuel de Windows Explorer, 
	il n'y a pas d'exécutable que vous puissiez lancer pour l'appeler.
	Pour l'utiliser, ouvrez simplement dans Windows Explorer un dossier 
	contenant des images et faites un clic droit sur le fichier choisi. 
        Une miniature apparaîtra directement dans le menu contextuel. 

========================================================================

Exigences système

* Windows 95/98/ME, Windows NT/2000/XP ou supérieur
* Environ 2 Mo d'espace disque
