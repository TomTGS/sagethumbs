LocalizeRC - A tool for localizing/translating Resource Scripts (RC files).

========================================================================

Usage:

	LocalizeRC.exe [/Automation] [project.lws]
	
Parameters:

	/Automation - Automatically preform next steps:
		1) Load project file (specified in command-line or from registry);
		2) Click "Create/Actualize INI from Input RC";
		3) Click "Create Output RC with INI";
		4) Quit.

	project.lws - Loads this project, otherwise project loaded from registry.

========================================================================

System Requirements:

  * Windows 95/98/ME/NT/2000/XP/2003/Vista/2008/7
  * About 1 Mb of disk space
